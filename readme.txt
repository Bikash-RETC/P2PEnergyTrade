
On Cloud

Install Python
Install Flask
Install PyMongo
Install MongoDb on Cloud

If venv creation required, then use virtualenv install -v requirement.txt
To launch the web app, launch webapp.sh


On EcoBox
Install Python
Install NetInterface
Install GPIO (for LED)
Check the switchstate.ini for the status
Check and edit initconf.ini for URLs 
To launch the EcoBox app, launch launchapp.sh
